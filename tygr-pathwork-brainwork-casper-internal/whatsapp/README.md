# WhatsApp Messages — Rasheq <> Taufiq

This folder is a placeholder for WhatsApp message history between Rasheq Rahman and Taufiq Zaman related to TYGR Labs, BrainWrk, PathWork, and Casper Academy.

## How to Export

WhatsApp messages cannot be pulled programmatically. To populate this folder:

1. Open WhatsApp (mobile or desktop)
2. Go to the chat with Taufiq Zaman
3. Tap the three dots (⋮) → More → Export chat
4. Choose "Without media" for text-only or "Include media" for full export
5. Save the exported `.txt` file to this folder
